Template Configuration Help

The template configuration is a JSON object that defines how bots created from this template will be configured.

Required Fields:
  • name: Default bot name
  • description: Default bot description
  • messaging: Platform (telegram, discord, etc.)
  • llm: AI model to use
  • prompt: System prompt for the AI
Optional Fields:
  • commands: Array of bot commands
  • settings: AI model settings
  • webhook_url: Webhook endpoint
  • tags: Array of tags
